# GraphStore Providers
PrivateGPT supports [FalkorDB](https://app.falkordb.cloud/) | [Neo4J](https://neo4j.com/).

In order to select one or the other, set the `graphstore.database` property in the `settings.yaml` file to `neo4j`.

```yaml
graphstore:
  database: falkordb or neo4j
```

## Neo4j

FalkorDB and Neo4j are graph database management systems that provide efficient and scalable solutions for storing and querying graph data.

### Configuration

To configure the database as the graph store provider, specify the following parameters in the `settings.yaml` file:

```yaml
graphstore:
  database: falkordb or neo4j

For falkordb:

For neo4j:
  url: neo4j://localhost:7687
  username: neo4j
  password: password
  database: neo4j
```

- **url**: The URL of the Neo4j server.
- **username**: The username for accessing the Neo4j database.
- **password**: The password for accessing the Neo4j database.
- **database**: The name of the Neo4j database.
